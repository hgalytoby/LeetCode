You are given an integer array  `nums`.

The  **digit range**  of an integer is defined as the difference between its  **largest**  digit and  **smallest**  digit.

For example, the digit range of 5724 is  `7 - 2 = 5`.

Return the sum of all integers in  `nums`  whose  **digit range**  is equal to the  **maximum digit range**  among all integers in the array.

**Example 1:**
```
Input:  nums = [5724,111,350]
Output:  6074
```

**Example 2:**
```
Input:  nums = [90,900]
Output:  990
```

**Constraints:**

-   `1 <= nums.length <= 100`
-   `10 <= nums[i] <= 105`


### Python
```py
class Solution:
    def maxDigitRange(self, nums: list[int]) -> int:
        result = diff = 0
        for num in nums:
            min_v, max_v = 9, 0
            for s_n in str(num):
                n = int(s_n)
                min_v = min(min_v, n)
                max_v = max(max_v, n)
            v = max_v - min_v
            if v > diff:
                result = num
                diff = v
            elif diff == v:
                result += num
        return result
```

public class Solution {
    public int MaxDigitRange(int[] nums) {
        int result = 0, diff = 0;
        foreach (int num in nums)
        {
            int minV = 9, maxV = 0;
            foreach (byte sN in num.ToString())
            {
                int n = sN - '0';
                minV = Math.Min(minV, n);
                maxV = Math.Max(maxV, n);
            }
            int v = maxV - minV;
            if (v > diff)
            {
                result = num;
                diff = v;
            } else if (diff == v)
            {
                result += num;
            }
        }
        return result;
    }
}
