You are given an integer  `n`.

Return  `true`  if its binary representation contains  **exactly one adjacent pair**  of set bits, and  `false`  otherwise.

**Example 1:**
```
Input:  n = 6
Output:  true
Explanation:
-   Binary representation of 6 is  `110`.
-   There is exactly one adjacent pair of set bits (`"11"`). Thus, the answer is  `true`.
```

**Example 2:**
```
Input:  n = 5
Output:  false
Explanation:
-   Binary representation of 5 is  `101`.
-   There is no adjacent pair of set bits. Thus, the answer is  `false`.
```

**Constraints:**

-   `0 <= n <= 105`


### Python
```py
class Solution:
    def consecutiveSetBits(self, n: int) -> bool:
        flag = False
        v = bin(n)[2:]
        for i in range(len(v) - 1):
            if f"{v[i]}{v[i + 1]}" == "11":
                if flag:
                    return False
                flag = True
        return flag
```

### Go
```go
func consecutiveSetBits(n int) bool {
	v := fmt.Sprintf("%b", n)
	flag := false
	for i := 0; i < len(v)-1; i++ {
		if v[i] == '1' && v[i+1] == '1' {
			if flag {
				return false
			}
			flag = true
		}
	}
	return flag
}
```

```c#
public class Solution {
    public bool ConsecutiveSetBits(int n) {
        string v = Convert.ToString(n, 2);
        bool flag = false;
        for (int i = 0; i < v.Length - 1; i++)
        {
            if (v[i] == '1' && v[i + 1] == '1')
            {
                if (flag) {
                    return false;
                }
                flag = true;
            }
        }
        return flag;
    }
}
```
