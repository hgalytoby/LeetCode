You are given an  `m x n`  grid. A robot starts at the top-left corner of the grid  `(0, 0)`  and wants to reach the bottom-right corner  `(m - 1, n - 1)`. The robot can move either right or down at any point in time.

The grid contains a value  `coins[i][j]`  in each cell:

-   If  `coins[i][j] >= 0`, the robot gains that many coins.
-   If  `coins[i][j] < 0`, the robot encounters a robber, and the robber steals the  **absolute**  value of  `coins[i][j]`  coins.

The robot has a special ability to  **neutralize robbers**  in at most  **2 cells**  on its path, preventing them from stealing coins in those cells.

**Note:**  The robot's total coins can be negative.

Return the  **maximum**  profit the robot can gain on the route.

**Example 1:**
```
Input:  coins = [[0,1,-1],[1,-2,3],[2,-3,4]]
Output:  8
Explanation:

An optimal path for maximum coins is:

1.  Start at  `(0, 0)`  with  `0`  coins (total coins =  `0`).
2.  Move to  `(0, 1)`, gaining  `1`  coin (total coins =  `0 + 1 = 1`).
3.  Move to  `(1, 1)`, where there's a robber stealing  `2`  coins. The robot uses one neutralization here, avoiding the robbery (total coins =  `1`).
4.  Move to  `(1, 2)`, gaining  `3`  coins (total coins =  `1 + 3 = 4`).
5.  Move to  `(2, 2)`, gaining  `4`  coins (total coins =  `4 + 4 = 8`).
```

**Example 2:**
```
Input:  coins = [[10,10,10],[10,10,10]]
Output:  40
Explanation:

An optimal path for maximum coins is:

1.  Start at  `(0, 0)`  with  `10`  coins (total coins =  `10`).
2.  Move to  `(0, 1)`, gaining  `10`  coins (total coins =  `10 + 10 = 20`).
3.  Move to  `(0, 2)`, gaining another  `10`  coins (total coins =  `20 + 10 = 30`).
4.  Move to  `(1, 2)`, gaining the final  `10`  coins (total coins =  `30 + 10 = 40`).
```

**Constraints:**

-   `m == coins.length`
-   `n == coins[i].length`
-   `1 <= m, n <= 500`
-   `-1000 <= coins[i][j] <= 1000`


### Python
```py
class Solution:
    direction = [(0, 1), (1, 0)]

    def maximumAmount(self, coins: List[List[int]]) -> int:
        n, m = len(coins), len(coins[0])
        result = -math.inf
        queue = deque([(0, 0, coins[0][0], 0)])
        memo = {(0, 0, 0): coins[0][0]}

        if coins[0][0] < 0:
            queue.append((0, 0, 0, 1))
            memo = {(0, 0, 1): 0}

        while queue:
            cur_x, cur_y, cur_coin, robbers = queue.popleft()

            if cur_coin < memo.get((cur_x, cur_y, robbers), -math.inf):
                continue

            if cur_x == n - 1 and cur_y == m - 1:
                result = max(result, cur_coin)
                continue

            for x, y in self.direction:
                next_x, next_y = cur_x + x, cur_y + y

                if next_x >= n or next_y >= m:
                    continue

                next_coin = cur_coin + coins[next_x][next_y]
                if next_coin > memo.get((next_x, next_y, robbers), -math.inf):
                    memo[(next_x, next_y, robbers)] = next_coin
                    queue.append((next_x, next_y, next_coin, robbers))

                if (
                    coins[next_x][next_y] < 0
                    and robbers < 2
                    and cur_coin > memo.get((next_x, next_y, robbers + 1), -math.inf)
                ):
                    memo[(next_x, next_y, robbers + 1)] = cur_coin
                    queue.append((next_x, next_y, cur_coin, robbers + 1))
        
        return result
```
