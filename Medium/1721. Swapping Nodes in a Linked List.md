You are given the  `head`  of a linked list, and an integer  `k`.

Return  _the head of the linked list after  **swapping**  the values of the_ `kth`  _node from the beginning and the_ `kth`  _node from the end (the list is  **1-indexed**)._

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/09/21/linked1.jpg)
```
Input: head = [1,2,3,4,5], k = 2
Output: [1,4,3,2,5]
```

**Example 2:**
```
Input: head = [7,9,6,6,7,8,3,0,9,5], k = 5
Output: [7,9,6,6,8,7,3,0,9,5]
```

**Constraints:**

-   The number of nodes in the list is  `n`.
-   `1 <= k <= n <= 105`
-   `0 <= Node.val <= 100`


### Python
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def swapNodes(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        array = []
        while head.next:
            array.append(head)
            head = head.next
        array.append(head)
        array[k - 1].val, array[-k].val = array[-k].val, array[k - 1].val
        return array[0]
```

# Definition for singly-linked list.

#         self.val = val
#         self.next = next
class Solution:
    def swapNodes(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        l = r = head
        for i in range(k - 1):
            l = l.next
        tail = l
        while tail.next:
            tail, r = tail.next, r.next
        l.val, r.val = r.val, l.val
        return head
