You are given a positive integer  `n`  representing  `n`  cities numbered from  `1`  to  `n`. You are also given a  **2D**  array  `roads`  where  `roads[i] = [ai, bi, distancei]`  indicates that there is a  **bidirectional** road between cities  `ai`  and  `bi`  with a distance equal to  `distancei`. The cities graph is not necessarily connected.

The  **score**  of a path between two cities is defined as the  **minimum** distance of a road in this path.

Return  _the  **minimum** possible score of a path between cities_ `1` _and_ `n`.

**Note**:

-   A path is a sequence of roads between two cities.
-   It is allowed for a path to contain the same road  **multiple**  times, and you can visit cities  `1`  and  `n`  multiple times along the path.
-   The test cases are generated such that there is  **at least**  one path between  `1`  and  `n`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2022/10/12/graph11.png)
```
Input: n = 4, roads = [[1,2,9],[2,3,6],[2,4,5],[1,4,7]]
Output: 5
Explanation: The path from city 1 to 4 with the minimum score is: 1 -> 2 -> 4. The score of this path is min(9,5) = 5.
It can be shown that no other path has less score.
```

**Example 2:**

![](https://assets.leetcode.com/uploads/2022/10/12/graph22.png)
```
Input: n = 4, roads = [[1,2,2],[1,3,4],[3,4,7]]
Output: 2
Explanation: The path from city 1 to 4 with the minimum score is: 1 -> 2 -> 1 -> 3 -> 4. The score of this path is min(2,2,4,7) = 2.
```

**Constraints:**

-   `2 <= n <= 105`
-   `1 <= roads.length <= 105`
-   `roads[i].length == 3`
-   `1 <= ai, bi <= n`
-   `ai != bi`
-   `1 <= distancei <= 104`
-   There are no repeated edges.
-   There is at least one path between  `1`  and  `n`.


### Python
```py
class Solution:
    def minScore(self, n: int, roads: List[List[int]]) -> int:
        def dfs(num):
            for node in group[num]:
                if node not in visited:
                    visited.add(node)
                    dfs(node)

        max_v = math.inf
        group = defaultdict(list)
        s = {}
        visited = {1}
        for x, y, z in roads:
            group[x].append(y)
            group[y].append(x)
            s[x] = min(s.get(x, max_v), z)
            s[y] = min(s.get(y, max_v), z)
            
        dfs(1)
        return min(s[v] for v in visited)
```

### Go
```go
func minScore(n int, roads [][]int) int {
    var dfs func(int)
    result := math.MaxInt
    group := make(map[int][]int)
    s := make(map[int]int)
    visited := make(map[int]struct{})
    visited[1] = struct{}{}
    for _, item := range roads {
        x, y, z := item[0], item[1], item[2]
        group[x] = append(group[x], y)
        group[y] = append(group[y], x)
        if _, ok := s[x]; ok {
            s[x] = min(s[x], z)
        } else {
            s[x] = z
        }
        if _, ok := s[y]; ok {
            s[y] = min(s[y], z)
        } else {
            s[y] = z
        }
    }
    dfs = func(num int) {
        for _, node := range group[num] {
            if _, ok := visited[node]; !ok {
                visited[node] = struct{}{}
                dfs(node)
            }
        }
    }
    dfs(1)
    for k, _ := range visited {
        result = min(result, s[k])
    }
    return result
}
```

### C#
```c#
public class Solution {
    public int MinScore(int n, int[][] roads) {
        int result = int.MaxValue;
        var group = new Dictionary<int, List<int>>();
        var s = new Dictionary<int, int>();
        var visited = new HashSet<int> { 1 };
        foreach (var item in roads) {
            int x = item[0], y = item[1], z = item[2];
            if (!group.ContainsKey(x))
            {
                group[x] = new List<int>();
            } 
            if (!group.ContainsKey(y))
            {
                group[y] = new List<int>();
            } 
            group[x].Add(y);
            group[y].Add(x);
            s[x] = s.ContainsKey(x) ? Math.Min(s[x], z) : z;
            s[y] = s.ContainsKey(y) ? Math.Min(s[y], z) : z;
        }
        void Dfs(int num) {
            foreach (var node in group[num]) {
                if (!visited.Contains(node)) {
                    visited.Add(node);
                    Dfs(node);
                }
            }
        }
        Dfs(1);
        foreach (var k in visited) {
            result = Math.Min(result, s[k]);
        }
        return result;
    }
}
```
